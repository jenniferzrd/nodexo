Installation des plugins via NPM

On initialise un projet nodejs avec la commande npm init
UN fichier de configuration package.json est généré
ce fichier est le patron, le plan, de chaque proket, il permet de stocker des plugins et leurs versions, des scripts, etc...

UNe fois le projet initialisé, on peut installer des modules avec la commande npm install nom-du-module
Par ex npm install vue

Ce fichier json est exportable, il permet de distribuer votre app facilement par ex via github
